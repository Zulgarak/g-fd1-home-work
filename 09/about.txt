1) Написать стили в style.css для index.html по макетам layout-big-size.png и layout-small-size.png. Размер блоков должны быть адаптивными по размеру окна.

Статические размеры для блоков 100px. Отступы для блоков 5px.

2) Сверстать кастомные чекбокс внутри .main-content блока. Стиль - ваша фантазия. Чекбокс должен иметь стили для двух состояний и для анимации навидения курсора на него.
